## Lưu trữ Frontend và Tích hợp API trên AWS
Trong workshop này, bạn sẽ học cách triển khai một **ứng dụng web frontend** trên AWS và **kết nối nó với API backend** được lưu trữ thông qua **Amazon API Gateway**.  
Hoạt động này là sự kết hợp giữa  **hosting frontend** và **tích hợp API**, cho thấy cách các dịch vụ AWS có thể hỗ trợ các ứng dụng web serverless tương tác.

**Amazon S3** – Lưu trữ và phục vụ các tài nguyên web tĩnh của bạn (HTML, CSS, JS).  
**Amazon CloudFront** – Phân phối website của bạn trên toàn cầu với HTTPS và độ trễ thấp.  
**Amazon API Gateway** – Cung cấp các endpoint API backend mà frontend của bạn có thể gọi.  

Workshop này minh họa cách kết nối một website tĩnh với backend API thông qua API Gateway, tạo nên một complete **kiến trúc web serverless** cho phép tương tác dữ liệu thời gian thực giữa frontend và backend.
